## 費用

| 項目 | 金額 |
|------|------|
| 基本出願料 (Micro Entity) | $65 |
| 100シート超過料（補足164ページ分） | $90 |
| **合計** | **$155** |

※ メインの21ページのみ提出する場合は **$65** のみ

## 手順

1. Patent Center にアクセス → アカウント作成/ログイン
2. 「New Submission」→「Provisional Application」
3. SB/16 の情報を入力（`SB16-cover-sheet.md` 参照）
4. Micro Entity を選択 → SB/15A の認証にチェック
5. PDF をアップロード
6. 支払い → 提出
7. **Acknowledgement Receipt を保存**（出願番号と出願日が記載される）

## 提出後

- 出願番号と出願日をメモ
- 12ヶ月以内（2027-03-19まで）に:
  - USPTO Non-Provisional に転換
  - JPO に日本語版を出願（優先権主張）
